# **App Name**: 書道教室Schedule

## Core Features:

- 生徒ログイン/ログアウト: Firebase Authenticationを使用して、生徒がメールアドレスとパスワードでログイン/ログアウトできるようにします。
- 生徒ダッシュボード: 生徒は、自分の今後の授業予定を一覧で確認できます。カレンダー表示では、授業がある日に印が表示され、タップすると詳細が表示されます。
- 振替申請: 生徒は、対象授業を選び、希望日候補（2〜3個）とメモを送信できます。
- お知らせ閲覧: 公開済みのお知らせの一覧を閲覧し、最新3件をマイページ上部に表示します。
- 管理者ログイン/ログアウト: 管理者は、メールアドレスとパスワードでログイン/ログアウトできます。
- 月間スケジューラ: 管理者は、土日の固定時間枠グリッドを表示し、生徒を割り当てることができます。定員超過は不可とします。
- 生徒管理: 生徒の名前・コース（月2/月3）・メールアドレスを管理します。
- 振替申請管理: 振替申請を承認/却下します。
- お知らせ投稿: タイトル・本文・公開/下書きを設定して、お知らせを投稿します。
- セキュリティルール: 未ログインユーザーはアクセス不可とし、生徒は自分のデータのみ閲覧可能とします。管理者は全データのread/writeを許可します。Firestore Security Rulesで担保します。

## Style Guidelines:

- プライマリーカラー：薄い藍色 (#E6E9F2) をベースに、上品で落ち着いた印象を与えます。
- 背景色：白 (#FFFFFF) を使用し、清潔感と余白を強調します。
- アクセントカラー：朱色 (#D9534F) を使用し、重要な要素やアクションを際立たせます。ただし、派手になりすぎないように、彩度を抑えます。
- 日本語UIのため、全体的に上品な雰囲気を持つ'PT Sans' (sans-serif)をボディテキストに採用します。 可読性を重視し、本文や注釈に利用します。タイトルや見出しなど、より強い印象を与えたい箇所には'Poppins' (sans-serif)を使用します。
- Note: currently only Google Fonts are supported.
- シンプルなアイコンセットを使用し、UIの分かりやすさを向上させます。和のテイストを取り入れ、筆のタッチや墨の色合いを表現します。
- 余白を十分に確保し、要素間の適切な距離を保つことで、視覚的なノイズを減らし、コンテンツに集中できるようにします。モバイルファーストで、レスポンシブなレイアウトを実装します。
- 画面遷移やローディング時に、スムーズなアニメーションを使用し、ユーザーエクスペリエンスを向上させます。和のテイストに合わせた、控えめなアニメーションを心がけます。
